js-char

character code bricks for JavaScript

Installation

Can be managed through duo, component, bower, or npm.

let char = require( "aureooms-js-char" ) ;

Usage

char.chr( 97 ) ; // "a"
char.ord( "\n" ) ; // 10

char.ord( "M" ) ; // 77
char.ord( "Μ" ) ; // 924

char.islower( "µ" ) ; // true
char.isupper( "À" ) ; // true

char.upper( "µ" ) ; // Μ
char.lower( "À" ) ; // à
